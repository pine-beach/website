0.1 ‚Äî Verify OpenClaw is installed

openclaw --version

You should see a version number like openclaw/2026.x.x. If you see command not found, go to 0.2.

0.2 ‚Äî Install OpenClaw (if needed)

macOS or Linux:

curl -fsSL https://openclaw.ai/install.sh | bash

This downloads the CLI, installs it globally, and launches an onboarding wizard. Follow the wizard ‚Äî it will ask you to configure your AI provider.

Windows: OpenClaw requires WSL2. First install it (wsl --install in PowerShell as Administrator, restart), then open your WSL2 terminal and run the Linux command above.

‚ö†Ô∏è command not found after install: Run npm prefix -g to find where npm installs global packages, then add that bin folder to your PATH.

0.3 ‚Äî Verify your AI model is configured

openclaw models status

You should see at least one provider listed as active. If nothing is configured, run openclaw onboard and follow the wizard.

0.4 ‚Äî Verify the Gateway is running

openclaw gateway status

Should say running or active. If not: openclaw gateway start

The setup script does the heavy lifting ‚Äî asks a series of questions, then generates your entire workspace pre-filled with your company name and agent names.

2.1 ‚Äî Run the script

~/workspace is where your agent files will live. Change it to any path you like. The rest of this guide assumes ~/workspace.

2.2 ‚Äî What the script will ask

Can I skip Discord? Yes. Talk to agents via openclaw dashboard or the CLI: openclaw agent --agent ceo --message "your message". Discord is the designed experience but not required.

4.1
Create a Discord server
In Discord, click + in the sidebar ‚Üí Create My Own ‚Üí For me and my friends. Name it "[Your Company] HQ".
4.2
Create channels for each agent
One text channel per agent: #ceo-office, #dev-workshop, #marketing-hub, #finance-desk, #security-ops, #support-desk, #general, #war-room.
4.3
Create a Discord bot
Go to discord.com/developers/applications ‚Üí New Application ‚Üí name it after your CEO. Under Bot, enable Message Content Intent (required). Click Reset Token and copy it.
4.4
Enable Developer Mode & collect IDs
User Settings ‚Üí Advanced ‚Üí Developer Mode ON. Right-click server icon ‚Üí Copy Server ID. Right-click your username ‚Üí Copy User ID. Save both.
4.5
Add bot token to OpenClaw
openclaw config set channels.discord.token '"YOUR_BOT_TOKEN"' --json openclaw config set channels.discord.enabled true --json openclaw gateway restart
4.6
Invite the bot and pair
In Developer Portal ‚Üí OAuth2 ‚Üí URL Generator: check bot and applications.commands with Send Messages, Read Message History permissions. Paste invite URL in browser. Then DM your bot in Discord and run openclaw pairing approve discord CODE.
